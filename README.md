## Cara Menjalankan Kode
Karena kode JavaScript ini berjalan di browser (Client Side), ikuti langkah ini:

1.  Clone repo ini atau download sebagai ZIP.
2.  Buka folder project di **VS Code**.
3.  Buka file `index.html` di dalam folder materi yang ingin dicoba.
4.  Klik kanan -> **Open with Live Server**.
5.  Buka **Inspect Element (F12) -> Console** untuk melihat output program.
